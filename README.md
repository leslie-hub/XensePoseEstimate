# 项目描述

1、该姿态评估系统，利用可穿戴式传感器Xsens，获得用户姿态信息，并将姿态信息输入到我们设计好的姿态评估中，从而实现评估用户姿态状况的功能。

2、该系统包含数据库功能，便于查找用户的姿态评估信息

3、该系统从多个角度考虑用户姿态状况

4、该系统结合多种标准姿态对用户姿态状况进行评估
